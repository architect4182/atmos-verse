# ATMOSVERSE V2

## Project Vision

AtmosVerse is not a streaming platform.

AtmosVerse is a cinematic discovery operating system that helps users find movies, TV shows, anime, universes, moods, and curated journeys.

The experience should feel like:

* Apple TV+
* VisionOS
* Travel OS
* MUBI
* Netflix Editorial

The experience should NOT feel like:

* Netflix clone
* IMDB clone
* Dashboard
* Admin panel
* Generic movie website

---

# Design Philosophy

AtmosVerse is built around discovery.

Users should feel like they are exploring a premium entertainment universe rather than browsing a database.

Every screen should feel:

* Cinematic
* Premium
* Emotional
* Curated
* Spacious
* Immersive

Keywords:

Luxury
Glass
Depth
Motion
Blur
Layers
Editorial
Premium Product Design

---

# Visual Style

## Apple Inspired

Use:

* Large typography
* Spacious layouts
* Layered depth
* Soft shadows
* Frosted glass surfaces
* Smooth motion
* Minimal chrome

Avoid:

* Hard borders
* Heavy outlines
* Dashboard widgets
* Bootstrap components

---

# Glassmorphism System

Use subtle glass effects.

Cards:

Background:
rgba(255,255,255,0.06)

Backdrop Blur:
20px

Border:
1px solid rgba(255,255,255,0.08)

Shadow:
soft ambient shadows

Glass should feel elegant.

Never use extreme transparency.
